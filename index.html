PowerShell: Automating Active Directory User Creation

This script automates the creation of AD users, populating all necessary fields, including assigning them to specific groups.


# PowerShell script to create Active Directory user
Import-Module ActiveDirectory

$UserName = "johndoe"
$Password = ConvertTo-SecureString "Password123!" -AsPlainText -Force
$OU = "OU=Users,DC=example,DC=com"
$Groups = @("Group1", "Group2")

New-ADUser -Name $UserName -AccountPassword $Password -Path $OU -Enabled $true `
    -UserPrincipalName "$UserName@example.com" -GivenName "John" -Surname "Doe" `
    -DisplayName "John Doe" -EmailAddress "johndoe@example.com"

# Add user to specified groups
ForEach ($Group in $Groups) {
    Add-ADGroupMember -Identity $Group -Members $UserName
}

Python: Sending Automated Alerts via SendGrid

This Python script sends an automated email alert using the SendGrid API, a solution that I integrated into Azure automation scripts to replace SMTP alerts.


import os
from sendgrid import SendGridAPIClient
from sendgrid.helpers.mail import Mail

def send_alert_email(recipient, subject, content):
    message = Mail(
        from_email='admin@example.com',
        to_emails=recipient,
        subject=subject,
        html_content=f"{content}"
    )
    try:
        sg = SendGridAPIClient(os.environ.get('SENDGRID_API_KEY'))
        response = sg.send(message)
        print(f"Email sent to {recipient} with status code {response.status_code}")
    except Exception as e:
        print(f"Error sending email: {e}")

# Usage
send_alert_email('johndoe@example.com', 'Server Alert', 'CPU usage has exceeded the threshold.')
